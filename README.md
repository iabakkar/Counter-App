# 🔢 Counter App

A simple and interactive Counter Application built using HTML, CSS, and JavaScript.

## 🚀 Features

* ➕ Increment the counter
* ➖ Decrement the counter (does not go below 0)
* 🔄 Reset the counter to zero

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript (DOM Manipulation)

## 📸 Preview

Displays a number that updates dynamically when buttons are clicked.

## 💡 Learning

This project helps in understanding:

* Event Listeners 🎯
* Variables & State 📊
* DOM Manipulation 🧠

## ▶️ How to Use

1. Click **Increment** to increase number
2. Click **Decrement** to decrease number
3. Click **Reset** to set it back to 0
